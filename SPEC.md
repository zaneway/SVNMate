# SVNMate - macOS SVN 客户端

## 1. 项目概述

- **项目名称**: SVNMate
- **类型**: macOS 原生应用 (SwiftUI + AppKit)
- **核心功能**: 基于本地 svn 命令行的图形化 SVN 客户端，支持仓库检出、更新、提交、对比等操作
- **目标用户**: 需要使用 SVN 进行版本控制的开发者

## 2. UI/UX 设计

### 窗口结构

- **主窗口** (NSWindow): 1200x800，最小 800x600
- **侧边栏**: 仓库列表 + 操作面板 (250pt 宽度)
- **主内容区**: 文件浏览器 + 操作详情

### 视觉设计

- **主色调**: 
  - 主色: #007AFF (系统蓝)
  - 背景: System background (支持明暗模式)
  - 强调: #34C759 (成功), #FF3B30 (错误), #FF9500 (警告)
  
- **字体**: SF Pro (系统字体)
  - 标题: 18pt semibold
  - 正文: 13pt regular
  - 代码/路径: SF Mono 12pt

- **间距**: 8pt 网格系统

### 页面结构

1. **欢迎页**: 无仓库时显示，引导检出或打开
2. **仓库详情页**: 侧边栏显示仓库信息 + 操作按钮，主区域显示文件树
3. **对比视图**: 展示本地与 SVN 版本的差异

### 组件

- **仓库卡片**: 显示仓库名、路径、最后更新时间
- **文件树**: 支持展开/折叠，显示文件状态图标 (M/A/D/?)
- **操作按钮**: 检出、更新、提交、对比、刷新
- **状态栏**: 显示当前操作进度和结果

## 3. 功能规范

### 核心功能

1. **仓库管理**
   - 从 URL 检出新仓库 (svn checkout)
   - 打开本地已检出的仓库
   - 记住最近打开的仓库
   - 删除仓库（仅移除记录，不删除文件）

2. **文件操作**
   - 浏览仓库文件树
   - 查看文件状态 (svn status)
   - 更新到最新版本 (svn update)
   - 提交更改 (svn commit)
   - 查看文件差异 (svn diff)

3. **辅助功能**
   - 查看提交日志 (svn log)
   - 查看版本信息 (svn info)
   - 清理锁定 (svn cleanup)

### 数据存储

- 使用 UserDefaults 存储:
  - 最近打开的仓库列表 (路径 + URL)
  - 用户偏好设置

### 错误处理

- SVN 命令执行失败时显示错误提示
- 网络超时处理
- 认证失败提示重新输入用户名密码

## 4. 技术架构

### 分层设计

```
┌─────────────────────────────────────┐
│           Views (SwiftUI)          │
├─────────────────────────────────────┤
│          ViewModels                │
├─────────────────────────────────────┤
│           Services                 │
│   - SVNService (命令行封装)         │
│   - RepositoryStore (仓库管理)      │
├─────────────────────────────────────┤
│           Models                   │
└─────────────────────────────────────┘
```

### 关键类

- **SVNService**: 执行 svn 命令，返回结果
- **Repository**: 仓库模型 (path, url, name, lastUpdate)
- **FileNode**: 文件树节点 (name, path, status, children)
- **RepositoryViewModel**: 仓库页面逻辑
- **MainViewModel**: 主应用逻辑
